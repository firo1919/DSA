## 1. Steps

1. Start at head.
    
2. Visit current node.
    
3. Move to `next`.
    
4. Repeat until end (`None`).
    

## 2. Example (Python) â€“ Convert Linked List to Array

```python
def convert_to_array(head):
    array = []
    current = head
    while current:
        array.append(current.value)
        current = current.next
    return array
```
