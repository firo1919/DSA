## 2. Definition / Idea

Binary Search can approximate real values within a precision threshold.

## 3. Example (Python)

```python
def binary_search_float(func, low, high, eps=1e-6):
    while high - low > eps:
        mid = (low + high) / 2
        if func(mid):
            high = mid
        else:
            low = mid
    return (low + high) / 2
```
