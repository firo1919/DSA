### Template 1 – Classic

```python
while low <= high:
    mid = (low + high) // 2
    if arr[mid] < target:
        low = mid + 1
    else:
        high = mid - 1
```

### Template 2 – Lower Bound Style

```python
while low < high:
    mid = (low + high) // 2
    if arr[mid] < target:
        low = mid + 1
    else:
        high = mid
```

### Template 3 – Upper Bound Style

```python
while low < high:
    mid = (low + high + 1) // 2
    if arr[mid] > target:
        high = mid - 1
    else:
        low = mid
```
