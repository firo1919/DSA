## 1. Definition / Idea

Computes cumulative sum of elements in a single array.

## 2. Example (Python)

```python
def prefix_sum(nums):
    prefix = [0] * len(nums)
    prefix[0] = nums[0]
    for i in range(1, len(nums)):
        prefix[i] = prefix[i-1] + nums[i]
    return prefix
```

## 3. Use Cases

- Range sum queries in O(1).
    
- Basis for advanced problems like pivot index.
